## ⚠️ قواعد مهمة من Google

1. **لا تذكر**:
   - عروض سعرية
   - وعود غير دقيقة ("الأفضل في العالم")
   - منافسين بالاسم
   - أسماء برامج أخرى لتسويق التطبيق

2. **اللغة الافتراضية إلزامية** (العربية في حالتنا) — البقية اختيارية

3. **حد الأحرف 500** — استخدمه بحكمة

4. **التحديث**: لكل إصدار جديد لازم release notes جديدة (versionCode يزيد)

## نصائح Pro

1. **استخدم emoji باعتدال** — يجذب الانتباه لكن لا تبالغ
2. **ابدأ بأهم ميزة** — أول سطر هو الأكثر مشاهدة
3. **اذكر فائدة المستخدم** ("توفر الوقت") لا الميزة التقنية ("لوحة بيانات")
4. **حدّث لكل إصدار** — حتى لو تغييرات صغيرة، اذكرها
